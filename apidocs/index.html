ZWL API Documentation

Introduction

Welcome to the ZWL API documentation. This API provides access to various services, including user management, crop marketplace, market price reports, and farmer data. Below is the comprehensive API reference for interacting with these services.

Base URL

https://apidocs.zowasel.com/v1

Authentication

All API requests require authentication via an API key. Obtain your API key from Zowasel by onboarding with our technical team and generating a key.

Header

Authorization: Bearer YOUR_API_KEY
